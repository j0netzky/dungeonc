This is DungeonC! Save the princess or else!

Buttons you will useful:

W = Forward
S = Backward
A = Left 
D = Right
Space = Jump
Enter = Interact
Esc = Quit (Only works in End screen so far: Alternatively, use Alt + f4)
M = Mute music (Will be added in future)

Made by Markku Harri & Joona Kela with Unity 5.2
